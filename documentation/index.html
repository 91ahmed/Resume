Configurations

No difficult configurations, just upload the files from the OResume folder to your PHP host and the website will work fine, You don't need to configure a database the website data is fully based on JSON.

Dashboard

How to access the dashboard?

Just write in the URL "dashboard" and if you want to change it for security open the project files and change the "dashboard" folder name with the name that you want.
